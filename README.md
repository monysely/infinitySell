## Istruzioni per Clone Repository

Da terminale:

- git clone UrlDellaRepo.
- composer install
- cp .env.example .env -> cloniamo il file env e lo impostiamo per il nostro ambiente locale
- php artisan key:gen

Pronti a lavorare.
